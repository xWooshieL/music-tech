# Презентация MusicTech — «Научный Телеграф» 2026

Доклад на 15–20 минут, **18 слайдов**. Стиль согласован со статьёй
(`article/main.tex`) и тезисами (`article/тезисы.tex`): чёрно-белая
научная типографика, Times-подобный serif, минималистичный beamer без
украшений.

## Структура

- [`presentation.tex`](presentation.tex) — точка входа, preamble +
  `\input{sections/...}`.
- [`sections/`](sections/) — 18 `.tex`-фрагментов, по одному на слайд.
- [`figures/tikz/`](figures/tikz/) — все TikZ-схемы и PGFPlots-графики,
  созданные специально под презентацию.
- [`build.ps1`](build.ps1) — два прогона `pdflatex`.

## Сборка

```powershell
cd article/presentation
.\build.ps1
```

или вручную:

```powershell
pdflatex -interaction=nonstopmode presentation.tex
pdflatex -interaction=nonstopmode presentation.tex
```

Результат — `presentation.pdf`.

## Стиль графики

- Чёрно-белая, как в `solve.ipynb`/`test.ipynb` (см. `notebooks/`
  оригинала).
- Шрифт: Times Roman через `mathptmx`.
- Графики через `pgfplots`, схемы через `tikz`. Никакого matplotlib —
  всё в LaTeX для консистентности с статьёй.
- Цвет акцента — серый и чёрный, RL-блоки выделены толстой обводкой,
  не цветом.

## Соответствие тезисам

Слайды 12–13 (предлагаемая архитектура + математика) воспроизводят
**РИС. 1** и формулу награды (1) из `article/тезисы.tex` буква-в-букву.
Это критично: внешний экспертный совет конференции будет сверять
презентацию с тезисами.

## Что брать из ноутбуков

Графики на слайдах 4, 5, 10 опираются на структуру данных из
`src/musictech-app/notebooks/02_score_format.ipynb`,
`03_performance_midi.ipynb`, `04_oltw.ipynb`, `05_hmm.ipynb`. Сами
данные перерисованы в LaTeX (через `pgfplots`) в чёрно-белом
научном стиле — без переключения backend'ов, без зависимости от
matplotlib.
